# 🚁 Mission Logger Drone

## Overview

mission_logger_drone is a drone software project built using Python and Microsoft AirSim.

The project demonstrates how to record and analyze autonomous drone missions by logging mission events to a persistent file.

The drone performs an autonomous patrol mission while recording important flight events such as takeoff, waypoint arrivals, scans, landing, and mission completion.

---

## Features

* Autonomous waypoint patrol
* Mission event logging
* Timestamped log entries
* Waypoint tracking
* Scan event tracking
* Landing detection
* Mission summary generation
* Flight duration calculation
